# Foodbank Maastricht Design Handoff

Figma cannot be generated as a native `.fig` file from this local static-site workspace, but the SVG files here are built to import into Figma.

Full-page editable SVG boards are in `design/pages/`:

- `home.svg`
- `about.svg`
- `get-involved.svg`
- `events.svg`
- `contact.svg`

There is also a combined multi-page board:

- `all-pages-full.svg`

These files include the page navigation, hero areas, content sections, forms, cards, social areas, map placeholders, illustrations, and footer.

How to use it:

1. Open Figma.
2. Drag `foodbank-maastricht-figma-import.svg` onto the canvas.
3. Use it as an editable visual handoff for palette, typography, home-page layout, and component direction.

For image swaps in the website, replace the SVG files in `assets/` with real images using the same filenames, or update the `src="assets/..."` paths in the HTML files.
